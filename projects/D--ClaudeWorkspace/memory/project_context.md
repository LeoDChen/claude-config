**当前状态（2026年4月）：**
- 阿里巴巴国际站店铺运营，主营婴儿纺织品（连体衣、套装、布尿裤、围兜、隔尿垫、吸汗巾、纱布包单、家居服、儿童纱布服装）
- 工厂规模：2015年成立，500万资本，6000㎡，200+工人，BSCI/ISO9001认证
- MOQ：500-1000件/款
- 两个P4P计划：日预算版 ¥175/天 + 智投版 ¥111/天
- 全店517个产品，其中87个在P4P推广，395个从未推广

**核心发现：**
- 日预算版效率是智投版的1.7倍（单商机成本 ¥23.4 vs ¥40.9）
- 当前地域偏好偏发达国家 → 带来B2C小零售买家，与工厂B2B定位不匹配
- 非洲/中东市场的贸易商买家更匹配工厂模式
- B2B标题词（Wholesale/OEM/Custom/Factory）是商机质量的关键

**目标市场（基于P4P地域偏好）：**
- 核心：苏丹、埃及、刚果民主共和国、赞比亚、科特迪瓦、沙特阿拉伯、阿联酋
- 辅助：美国、欧洲主要国家

**近期操作：**
- 完成全店产品推广状态总表（517品完整分类）
- 完成两计划对比分析及V2.0修正
- 积累P4P优化方法论（产品分层/地域调整/标题优化/全链路框架）
- 已淘汰33个低效品

**下一步计划：**
- 调整地域偏好（加非洲/中东国家）
- 智投版优胜品转入日预算版
- 从395个未推广池中分批轮测
