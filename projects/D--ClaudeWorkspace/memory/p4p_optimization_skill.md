# P4P 优化技能手册

**文件位置**: `C:\Users\Administrator\Desktop\P4P优化专用\P4P优化技能手册.md`

## 核心方法论摘要

### 两计划效率基准
- 日预算版单商机成本 ¥23.4，智投版 ¥40.9，**日预算版效率是智投版的 1.7 倍**
- 日预算版日均商机 7.6，智投版 3.0
- 日预算版 CTR 稳定约 3.5%，智投版近期下降

### 产品分层（核心决策规则）
- 商机 ≥ 8 → 核心爆品（预算倾斜）
- 商机 4-7 → 优质品（正常维持）
- 商机 1-3 → 边缘保留（观察）
- 商机 0 → 待淘汰
- CTR > 3% + 商机少 → 有潜力继续测
- 曝光 < 100 → 数据不足

### 关键避坑
1. **重点国家溢价不支持非洲国家** — 只能用地月偏好加
2. **两个计划均已设 L1+** — 不要再改人群偏好
3. **发达国家 L1+ ≠ B2B 买家** — 发达国家L1+仍以小零售商为主，与工厂 MOQ 500-1000 不匹配
4. **地域偏好**是当前最主要的优化杠杆

### 店铺数据基线
- 全店 517 个产品，P4P 推广 87 个，未推广 395 个（+35个查不到数据）
- 395 个未推广品中大部分有自然搜索曝光

### 标题 B2B 关键词
高转化词: Wholesale, OEM/ODM, Custom, Bulk, Factory, Manufacturer, Supplier
